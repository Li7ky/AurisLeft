# AurisLeft

自用桌面音乐播放器 · **v1.0.0 正式版**

边听边改，稳定够用就继续用。

## 能做什么

- **搜歌**：多平台聚合搜索（网易 / 酷我 / 酷狗 / QQ）
- **播放**：队列、随机 / 循环；取链失败自动换源、降音质；支持连续切歌
- **整理**：收藏、最近播放、歌单
- **本地 / 下载**：扫本地文件夹，也能把歌存到电脑
- **小功能**：关窗进托盘、睡眠定时、深色 / 明亮主题、数据备份、检查更新

## 下载

Windows 安装包 → [Releases](https://github.com/Li7ky/AurisLeft/releases)

## 首次使用

1. 安装并打开 AurisLeft  
2. **设置** → 填写并开启 **西瓜糖 QQ 解析**密钥（播放主路径）  
3. 搜索点歌即可  

未配置密钥时仍可搜歌、扫本地；在线取链会提示去设置开启。

## 开发

```bash
cd zuoer
npm install
npm run dev      # 开发
npm run dist     # 打 Windows 安装包
```

## 数据与日志

- 用户数据：Electron `userData`（歌单 / 收藏 / 设置 / 密钥偏好等）
- 备份：设置 → 数据与诊断 → 导出 / 导入
- 日志：设置 → 打开日志目录（`userData/logs`）

## 说在前面

- 自用项目，主要为自己听歌方便
- 个别歌播不了多半是版权或音源限制，会自动尝试换源；仍失败就换一首
- 在线播放依赖西瓜糖 QQ 解析；洛雪兼容音源可作为扩展备用
