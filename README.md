# AurisLeft

自用桌面音乐播放器 · 当前 **测试版**（`0.2.0-beta`）

边听边改，功能还在打磨，正式版以后再说。

## 能做什么

- **搜歌**：多平台聚合搜索，音源能开关，音质能换
- **播放**：队列、随机 / 循环，该有的控制都在
- **整理**：收藏、最近播放、歌单
- **本地 / 下载**：扫本地文件夹，也能把歌存到电脑
- **小功能**：关窗进托盘、睡眠定时、深色 / 明亮主题

## 下载

Windows 安装包 → [Releases](https://github.com/Li7ky/AurisLeft/releases)

## 说在前面

- 自用项目，主要为自己听歌方便，不对外承诺稳定与支持
- 测试版内容可能随时改，遇到怪问题很正常
- 个别歌播不了多半是版权或音源限制，换一首就行
- 第一次打开若暂时搜不到，稍等几秒让音源热热身
