## 在线刷新镜像（与 README 一致）

原始链接示例：
```
https://raw.githubusercontent.com/pdone/lx-music-source/main/flower/latest.js
```

加速链接示例：
```
https://ghproxy.net/raw.githubusercontent.com/pdone/lx-music-source/main/flower/latest.js
```

其它加速站（替换 `https://ghproxy.net/` 前缀）：
- https://gh.llkk.cc/
- https://github.moeyy.xyz/
- https://ghproxy.cn/
- https://gh.api.99988866.xyz/
- https://ghp.ci/

另有 jsDelivr：`https://cdn.jsdelivr.net/gh/pdone/lx-music-source@main/`

应用启动时先读本地缓存，再按上述镜像依次尝试刷新。
